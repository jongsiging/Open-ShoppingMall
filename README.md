# Open-ShoppingMall


해당 프로젝트는 기존 쇼핑몰에 판매자를 추가하여 오픈 쇼핑몰를 구현하였습니다.
그 외에도 상단 메뉴를 동적으로 구현하였고, 상품 오래된순, 판매량순 정렬 등을 구현하였습니다.

## 사용 기술

- java
- jsp/servlet
- Spring Framework
- oracle
- 웹표준(HTML5, CSS, javascript)
- jQuery
- Ajax
- bootstrap
- git(+github)
